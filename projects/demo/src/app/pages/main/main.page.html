Dev Demo используется для визуализации наработок по текущим и новым компонентам.

Запуск в режиме разработки

  • Запустить сборку библиотек командой
  • В директории библиотеки projects/ng-ui-kit выполнить команду
  • Запустить demo командой ng serve demo
  • В директории demo projects/demo запустить команду

Правила разработки:

  • Создать новый компонент командой npm run "generate:ui-kit components/component-name"
  • Создать новый standalone компонент в demo командой "generate:demo pages/component-name"
  • После завершения разработки и тестирования добавить страницу с описанием нового компонента в корневой проект
